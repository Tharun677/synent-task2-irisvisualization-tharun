# Iris Data Visualization

## About the Project

This project was completed as part of my Data Science Internship at Synent Technologies.

The objective of this project was to visualize the Iris dataset using different types of charts and identify patterns among flower species.

## Dataset

Iris Dataset

Source:
https://www.kaggle.com/datasets/uciml/iris

## Visualizations Created

- Bar Chart
- Histogram
- Scatter Plot
- Feature Comparison using Pair Plot

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights

- The dataset contains three flower species.
- The dataset is balanced with equal observations for each species.
- Petal measurements help distinguish species more clearly than sepal measurements.
- Setosa is clearly separated from the other two species.
